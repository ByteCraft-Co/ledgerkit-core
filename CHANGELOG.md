- Core domain models (Money, Transaction, Category, Budget, Recurrence, typed IDs)
- Analytics (category breakdowns, monthly totals, budget progress)
- Rules engine with default auto-categorization and validation
- Storage abstraction with in-memory implementation
- Import/export for JSON snapshots and CSV transactions
- Utilities: decimal and date helpers, lightweight Result type
